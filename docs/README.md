## 📖 项目简介

导盲犬（Guide Dog）是一个创新的桌面AI助手，以可爱的金毛导盲犬形象出现在用户桌面上。它的核心创新在于**屏幕感知能力** - AI能够"看到"用户的屏幕内容，从而提供更智能、更贴合场景的辅助。

### 核心特性

🐕 **可爱的桌面宠物**
- 金毛导盲犬形象
- 静态图标，简洁友好
- 可拖拽到任意位置

👁️ **屏幕感知能力**
- 理解用户当前屏幕内容
- 基于视觉上下文回答问题
- 支持粘贴板截图识别

💬 **智能对话交互**
- 点击宠物或快捷键（Cmd+Shift+A）唤醒
- 支持多轮对话记忆
- Markdown格式回答，代码高亮

🔒 **隐私友好**
- 按需截图，用户完全可控
- 不做主动监控
- 数据不持久化

## 🚀 快速开始

### 系统要求
- macOS 10.15+
- 屏幕录制权限
- 粘贴板访问权限

### 安装使用
1. 下载并安装导盲犬
2. 首次启动配置 IdeaLab API 密钥
3. 点击小狗或按 Cmd+Shift+A 开始对话

### 基本操作
- **唤醒对话**：点击桌面小狗 或 按 Cmd+Shift+A
- **发送消息**：输入问题后点击发送
- **包含截图**：勾选"包含当前截图"选项
- **使用粘贴板**：勾选"粘贴板截图"选项
- **复制回答**：点击回答下方的复制按钮
- **新建会话**：关闭对话窗口后重新打开

## 🛠️ 技术栈

- **框架**：Electron
- **UI**：React + TypeScript + Tailwind CSS
- **AI服务**：IdeaLab（千问多模态模型 qwen-vl-max-latest）
- **API调用**：OpenAI TypeScript SDK
- **图片处理**：Base64 编码，直接发送原图
- **平台**：macOS

## 🔐 隐私与安全

### 数据处理原则
- ✅ 截图按需生成，用完即删
- ✅ 对话历史不持久化
- ✅ API密钥本地加密存储
- ✅ 用户完全控制数据流

### 权限说明
- **屏幕录制权限**：用于截取屏幕内容
- **粘贴板访问权限**：用于读取粘贴板截图
- **网络权限**：仅用于调用千问API

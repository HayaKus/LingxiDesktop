# 用户更新流程说明

## 📱 用户如何检测和更新应用

### 第一步: 打开设置

1. 启动"灵析"应用
2. 点击右上角的 **"⚙️ 更多"** 按钮

### 第二步: 检测更新

1. 在设置页面向下滚动
2. 找到 **"🔄 检测更新"** 区域
3. 点击 **"检测更新"** 按钮

### 第三步: 查看结果

**情况 A: 有新版本**
```
🎉 发现新版本 v0.2.0

发布日期: 2026-01-21

更新内容:
• 新增: 应用内检测更新功能
• 优化: 改进用户界面
• 修复: 若干已知问题

[立即下载]  [稍后提醒]
```

**情况 B: 已是最新版本**
```
✅ 已是最新版本

当前版本: 0.1.0
```

**情况 C: 检测失败**
```
❌ 检测失败
网络错误: 无法连接到更新服务器
```

### 第四步: 下载新版本

1. 点击 **"立即下载"** 按钮
2. 浏览器会自动打开下载页面
3. 下载新版本安装包
4. 安装新版本

## ❓ 常见问题

### Q: 需要注册账号吗?
**A:** 不需要!完全免费,无需注册任何账号。

### Q: 需要付费吗?
**A:** 不需要!更新检测和下载都是免费的。

### Q: 是自动更新吗?
**A:** 不是自动的,需要你手动点击"检测更新"按钮,如有新版本再点击下载。

### Q: 检测失败怎么办?
**A:** 
1. 检查网络连接是否正常
2. 稍后重试
3. 联系开发者获取最新版本

### Q: 旧版本还能用吗?
**A:** 可以!更新是可选的,不更新也不影响使用。

### Q: 更新后数据会丢失吗?
**A:** 不会!应用数据都保存在本地,更新不会影响已有数据。

## 🔒 安全说明

- ✅ 更新检测只读取版本信息,不会上传任何数据
- ✅ 下载链接直接来自官方发布渠道
- ✅ 安装包与直接从官网下载的完全一致
- ✅ 不会自动下载或安装,完全由用户控制

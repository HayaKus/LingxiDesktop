# 🚀 快速开始

本指南将帮助你快速安装和开始使用桌面伙伴。

## 📥 安装方式

### 方式一：Homebrew 安装（推荐）

Homebrew 是 macOS 上最流行的包管理器，使用它可以更方便地安装和更新应用。

#### 1. 安装 Homebrew（如果还没有）

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

#### 2. 添加 Tap 并安装

```bash
# 添加 Tap 仓库
brew tap HayaKus/iamdog

# 安装应用
brew install --cask iamdog
```

#### 3. 启动应用

安装完成后，可以从启动台或应用程序文件夹启动"桌面伙伴"。

#### 4. 更新应用

```bash
# 更新 Homebrew 索引
brew update

# 升级到最新版本
brew upgrade --cask iamdog
```

### 方式二：手动下载安装

#### 1. 下载安装包

访问 [GitHub Releases](https://github.com/HayaKus/IamDog/releases) 页面，下载最新版本的 DMG 文件。

例如：`lingxi-0.1.5.dmg`

#### 2. 安装应用

1. 双击下载的 DMG 文件
2. 将"桌面伙伴"图标拖到"Applications"文件夹
3. 从应用程序文件夹启动

#### 3. 解决安全提示

首次打开时，如果遇到"已损坏"或"无法打开"的提示：

**方法一：使用命令行**

```bash
xattr -cr /Applications/桌面伙伴.app
```

**方法二：系统设置**

1. 打开"系统设置"
2. 进入"隐私与安全性"
3. 在"安全性"部分，点击"仍要打开"

详见：[常见问题-DMG损坏](./常见问题-DMG损坏.md)

#### 4. 检查更新

应用内置了自动更新检测功能：

1. 打开应用
2. 点击右上角的"检查更新"按钮
3. 如果有新版本，会提示下载

## 🎯 首次使用

### 1. 配置 API

首次启动应用后，需要配置 AI 服务：

1. 点击设置按钮
2. 输入你的 API Key
3. 选择模型（Claude、GPT 等）
4. 保存配置

### 2. 开始对话

配置完成后，就可以在输入框输入问题，开始与 AI 对话了。

### 3. 屏幕感知功能

桌面伙伴具备屏幕感知能力，可以分析屏幕内容并提供相关帮助。

## ❓ 常见问题

### Q: Homebrew 安装失败怎么办？

A: 尝试以下步骤：
```bash
# 更新 Homebrew
brew update

# 重试安装
brew reinstall --cask iamdog
```

### Q: 应用无法启动？

A: 执行清除属性命令：
```bash
xattr -cr /Applications/桌面伙伴.app
```

### Q: 如何完全卸载？

A: 使用 Homebrew：
```bash
brew uninstall --cask --zap iamdog
```

手动卸载：
1. 从应用程序文件夹删除"桌面伙伴.app"
2. 删除配置文件：`~/Library/Application Support/桌面伙伴`

### Q: 如何切换安装方式？

A: 可以先卸载当前安装，然后使用另一种方式重新安装。配置文件会保留。
